# Muddy

Muddy is a private Dragon Ball Z MUD focused on real-time combat, training-based growth, and Ki as Power Level.

## Workspace

- `apps/server`: game server
- `apps/client`: lightweight web client
- `packages/shared`: shared types and game constants
- `docs`: design notes and project blueprint
